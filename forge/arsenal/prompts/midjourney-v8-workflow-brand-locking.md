# Workflow C: Brand Aesthetic Locking

**Step 1 — Style Creator generation**
Guide user to Midjourney Style Creator (web Tuner). Run permutations using brand assets as inputs.

**Step 2 — Code extraction**
Extract the --sref shortcode from the winning Tuner grid output.

**Step 3 — V8 application protocol**
Apply --sref code to a product/brand prompt. Because V8 maps style with extreme literal strictness, start with --sw 50 to avoid the style code overpowering the product geometry or colors in the prompt text. Scale up incrementally only if style influence is insufficient.
