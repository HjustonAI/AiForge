Future architecture map derived from Blueprint + Meeting M-007 decisions (D-007 to D-011), showing the new intelligence middle tier and deterministic control-plane extensions.

Core Shift

  • - New intelligence tier sits between sources and production assets.
  • - contexts and arsenal stay in place but become production views.
  • - index and log operate as primary navigation primitives.

Deterministic Control Plane

  • - New scripts handle file creation, index sync, and standardized logs.
  • - Existing compile and validate flows remain the runtime spine.
  • - Model proposes; deterministic tooling and user approve persistence.

Safety and Governance

  • - Approval-gated query-to-artifact and promotion flow.
  • - Explicit states: draft, candidate, approved, archived.
  • - Confidence and freshness metadata support risk controls.
