FORGE v0.3 - Runtime Mode Pipelines

Operational flow diagram for Prompt mode, Distill mode, and Wiki Deploy mode, including approval gates and file outputs in the current Forge implementation.

Prompt Mode

  • - Triggered by prompt generation requests and forge:prompt.
  • - Compiles master + target context before prompt-smith generation.
  • - Uses arsenal references and optional save with quality rating.

Distill Mode

  • - Builds new target contexts from research material.
  • - Enforces structure and token checks before publishing.
  • - Feeds future prompt mode quality through updated context assets.

Wiki Deploy Mode

  • - Triggered by forge:wiki with working-directory awareness.
  • - Uses llm-wiki seed and context to build deploy plan.
  • - Writes only after explicit user approval at the gate.
